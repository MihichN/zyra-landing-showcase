# ZyraPayments Landing Showcase

Public-safe showcase of a marketing landing page for a crypto payment platform.

## Highlights

- Product positioning and conversion-oriented sections.
- Responsive frontend structure.
- Feature, pricing/benefits, and CTA blocks.
- Animation-ready UI composition.

## Tech Stack

- Next.js, React, TypeScript.
- Tailwind CSS, Radix UI.
- Framer Motion-style interaction patterns.

## Included

- `docs/architecture.md` - landing page section structure.
- `examples/feature-section/featureList.ts` - safe content model example.

## Not Included

Private marketing data, unreleased pricing, analytics tokens, internal strategy docs, and production environment files are excluded.
